# Weather App Vue

Weather forecast application built with Vue 3 and Vite. It uses Open-Meteo, so the project can be published on GitHub without exposing private API keys.

## English

### Features

- Interface available in English, Spanish, French, Chinese, German, Hindi, and Arabic
- City search with geocoding powered by Open-Meteo
- 7-day forecast with localized labels
- Ready for GitHub Pages deployment

### Development

```bash
npm install
npm run dev
```

### Production build

```bash
npm run build
```

### Deploy on GitHub Pages

1. Push the repository to GitHub.
2. Open repository Settings > Pages.
3. In Build and deployment, select GitHub Actions as the source.
4. The included workflow will build and publish the app automatically on pushes to main.
